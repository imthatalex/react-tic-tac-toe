# Tic Tac Toe with React
This program uses the React JavaScript library to build a simple tic tac toe game. It allows users to play the classic game of tic tac toe against a computer opponent.

### Game Features
Users can play against the computer
The game board is implemented using a 3x3 grid of buttons
The game state is managed using React's state and props
The game logic is implemented using JavaScript functions
The game ends when either the player or the computer wins or there is a draw
A reset button allows the player to start a new game

### Built With
React - JavaScript library for building user interfaces
npm - Package manager for JavaScript
